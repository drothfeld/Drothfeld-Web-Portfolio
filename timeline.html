I graduated from UC Santa Cruz with a BA in Computer Science a quarter early in March 2016. For my undergraduate capstone project, I built a web application called GTC (Guitar Tab Creator) using the Web2py framework. I implemented an interactive digital guitar that played the note a user presses on the neck, and then records the note into a TAB. The user is given sheet music editing tools that can be used with the interactive guitar to create songs. Songs can be saved/downloaded and commented on (GTC can also play the song back in audio form).
